Probability & Combinatorics

Basic Probability

Probability measures how likely an event is to occur, expressed as a number between 0 (impossible) and 1 (certain).

Basic Probability:

P(event) = Number of favourable outcomes / Total number of outcomes

Key Terms:

  • Sample Space (S): the set of all possible outcomes
  • Event: a subset of the sample space
  • Complement P(A'): probability that A does NOT occur = 1 - P(A)

Probability Trees

A probability tree diagrams out all possible outcomes across multiple events. Each branch represents one outcome, and the probability of each branch is written on it. To find the probability of a sequence of events, multiply along the branches.

Example: Two Coin Flips

      H (0.5) → HH: 0.5 × 0.5 = 0.25
   H (0.5)
      T (0.5) → HT: 0.5 × 0.5 = 0.25
Start
      H (0.5) → TH: 0.5 × 0.5 = 0.25
   T (0.5)
      T (0.5) → TT: 0.5 × 0.5 = 0.25

All four outcomes have equal probability of 0.25. They sum to 1 ✓

Conditional Probability

Conditional probability is the probability of event A occurring given that event B has already occurred.

Conditional Probability:

P(A | B) = P(A ∩ B) / P(B)

Read as: "Probability of A given B"

Example

A bag has 3 red and 2 blue balls. You draw one ball (red) and don't replace it. What is the probability the second ball is also red?

After drawing red: 2 red remain out of 4 total balls.

P(2nd red | 1st red) = 2/4 = 0.5

Independent events: If A and B are independent, P(A|B) = P(A). Knowing B happened tells you nothing about A. Example: two separate coin flips.

Permutations

A permutation is an arrangement where order matters. The number of ways to arrange r items chosen from n distinct items is written as nPr or P(n,r).

Permutation Formula:

nPr = n! / (n - r)!

Where n! = n × (n-1) × (n-2) × ... × 1

Example

In how many ways can 3 students be chosen as President, Vice-President, and Secretary from a group of 10?

10P3 = 10! / 7! = 10 × 9 × 8 = 720

Order matters because the roles are different.

Combinations

A combination is a selection where order does not matter. The number of ways to choose r items from n is written as nCr or C(n,r) or nCr.

Combination Formula:

nCr = n! / (r! × (n - r)!)

nCr = nPr / r!   (we divide out the r! arrangements that don't matter)

Example

How many ways can a committee of 3 be chosen from 10 students?

10C3 = 10! / (3! × 7!) = (10 × 9 × 8) / (3 × 2 × 1) = 720/6 = 120

Permutation vs Combination: The Key Question

  • Does the order of selection matter? → Use Permutation
  • Does only the group matter, not the order? → Use Combination
  • Passwords, race positions, ranked lists → Permutation
  • Teams, committees, lottery draws → Combination
  • Always: nCr ≤ nPr (combinations are fewer because order is ignored)
