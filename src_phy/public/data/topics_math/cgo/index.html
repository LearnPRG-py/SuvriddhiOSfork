Coordinate Geometry

The Cartesian Plane

The Cartesian plane is a 2D surface defined by two perpendicular axes: the horizontal x-axis and the vertical y-axis. Any point on this plane can be described by an ordered pair (x, y), called its coordinates.

Distance Between Two Points

The distance between points A(x1, y1) and B(x2, y2) is derived from the Pythagorean theorem.

Distance Formula:

d = √[(x2 - x1)2 + (y2 - y1)2]

Example

Find the distance between A(1, 2) and B(4, 6).

d = √[(4-1)2 + (6-2)2] = √[9 + 16] = √25 = 5

Midpoint of a Line Segment

The midpoint M of a segment joining A(x1, y1) and B(x2, y2) is the average of their coordinates.

Midpoint Formula:

M = ((x1 + x2)/2 , (y1 + y2)/2)

Section Formula

If point P divides the segment AB in the ratio m:n internally, its coordinates are:

Section Formula (Internal Division):

P = ((mx2 + nx1) / (m+n) , (my2 + ny1) / (m+n))

Note: The midpoint is a special case of the section formula where m = n = 1.

Slope of a Line

The slope (or gradient) of a line measures how steeply it rises or falls. It is the ratio of vertical change to horizontal change.

Slope:

m = (y2 - y1) / (x2 - x1)

A horizontal line has slope 0. A vertical line has undefined slope.

Equations of a Line

Slope-Intercept Form: y = mx + c, where m is slope and c is the y-intercept.

Two-Point Form: (y - y1) = m(x - x1)

Standard Form: ax + by + c = 0

Parallel and Perpendicular Lines

Key Rules

  • Two lines are parallel if their slopes are equal: m1 = m2
  • Two lines are perpendicular if the product of their slopes is -1: m1 × m2 = -1

Intersection of Two Lines

To find where two lines cross, solve their equations simultaneously. The solution (x, y) is the point of intersection.

Example

Find the intersection of y = 2x + 1 and y = -x + 4.

Set equal: 2x + 1 = -x + 4

3x = 3, so x = 1, y = 3

Intersection: (1, 3)

Distance from a Point to a Line

The perpendicular distance from point (x0, y0) to the line ax + by + c = 0 is:

d = |ax0 + by0 + c| / √(a2 + b2)

Polar Coordinates

Instead of using (x, y), polar coordinates describe a point using its distance from the origin (r) and the angle it makes with the positive x-axis (θ).

Converting between Cartesian and Polar:

  • x = r cos(θ)
  • y = r sin(θ)
  • r = √(x2 + y2)
  • θ = arctan(y/x)

When to use polar coordinates: Polar coordinates are especially useful for circles, spirals, and problems with rotational symmetry, where Cartesian coordinates would lead to complicated equations.

Example

Convert the Cartesian point (3, 3) to polar coordinates.

r = √(9 + 9) = √18 = 3√2

θ = arctan(3/3) = arctan(1) = 45°

Polar form: (3√2, 45°)
